# Machine Learning — Research-Graded Project & Concept Repository

## Objective
A structured, research-oriented collection of **machine learning concepts, mathematical foundations, experiments, and advanced implementations**, designed to demonstrate depth, rigor, and long-term relevance for research, internships, and startup credibility.

---

## LEVEL 1 — Mathematical & Statistical Foundations

### 1. Linear Algebra for Machine Learning
**Description:**  
Core linear algebra concepts used in ML models including vector spaces, eigenvalues, singular value decomposition (SVD), matrix norms, and projections.

**Project Ideas:**
- PCA implementation from scratch (NumPy only)
- Eigenvector and variance visualization
- PCA vs Autoencoder dimensionality reduction study

---

### 2. Probability Theory for Learning Systems
**Description:**  
Random variables, probability distributions, expectation, variance, entropy, KL divergence, and Bayesian inference.

**Project Ideas:**
- Distribution simulators (Gaussian, Bernoulli, Poisson)
- KL divergence visualizer
- Bayesian inference experiments

---

### 3. Optimization Algorithms
**Description:**  
Loss minimization techniques including Gradient Descent, SGD, Momentum, RMSProp, Adam, and convex vs non-convex optimization.

**Project Ideas:**
- Optimizer comparison on identical loss surfaces
- Loss landscape visualization
- Convergence failure case studies

---

### 4. Information Theory
**Description:**  
Entropy, mutual information, cross-entropy loss, and information bottleneck principles.

**Project Ideas:**
- Mutual information estimator
- Feature selection using information gain
- Cross-entropy vs MSE analysis
